## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Docker

First, you should check that your Docker app is running

```bash
# run docker compose
$ docker-compose up

# stop docker
$ docker-compose stop

# build docker image
$ docker-compose build
```
